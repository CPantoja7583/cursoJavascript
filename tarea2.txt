Actividad Práctica – Cálculo del Área de un Círculo con JavaScript
En esta actividad pondrás en práctica conceptos básicos de JavaScript y manipulación del DOM, aplicados a un ejercicio sencillo de matemáticas. El objetivo es que aprendas a interactuar con el usuario mediante cuadros de diálogo, realizar operaciones con el objeto Math y mostrar resultados tanto en consola, como en alertas y en la propia página web. Además, consolidarás el uso de prompt, console.log, window.alert y document.getElementById.

Descripción de la Tarea
Debes desarrollar una pequeña aplicación web que calcule el área de un círculo aplicando la fórmula:

Los pasos a realizar son:

Crear un archivo index.html con la estructura básica de un documento HTML y un <div> con un id para mostrar el resultado.

Crear un archivo app.js que:

Solicite al usuario el diámetro del círculo con prompt().

Calcule el radio como la mitad del diámetro.

Obtenga el área utilizando Math.PI y Math.pow(radio, 2).

Muestre el resultado en:

La consola (console.log()).

Una ventana emergente (window.alert()).

La página web usando document.getElementById(...).innerHTML.

Vincular app.js a index.html usando <script src="app.js"></script>.

⚠️ El archivo PDF que se adjunta contiene ejemplos visuales y explicaciones complementarias que debes revisar antes de comenzar la tarea.

30 a 40 minutos.

Subir el proyecto comprimido en un archivo .zip (incluyendo index.html y app.js), o

Subir las capturas de pantalla solicitadas en formato PDF.
